# Untangle 🕸️

A graph-untangling puzzle. Nodes are connected by edges, but they start in a tangled arrangement where many edges cross.

## How to Play

- **Drag nodes** to new positions
- **Red edges** = crossing another edge
- **Green edges** = clear (no crossing)
- Untangle the entire graph so **all edges are green**

## Tips

- Start by moving a node that has many crossing edges
- Try to spread nodes around the canvas edges
- Look for the "structure" hidden in the tangle — most graphs have a nice symmetric solution

## Levels

8 levels of increasing complexity — from 4 nodes (6 edges) up to 12 nodes (20 edges).
