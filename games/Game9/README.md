# Game9 — Gravity Maze

A sliding puzzle where you flip gravity to move a ball through a maze.

Set gravity to ↑ ↓ ← →, and the ball slides in that direction until it hits a wall. Collect all 💎 gems, then reach the 🚀 exit to clear the level.

- 10 hand-crafted levels of increasing difficulty
- Keyboard (arrow keys / WASD), on-screen buttons, and swipe gestures
- Undo support
- Pure HTML/CSS/JS, mobile friendly
